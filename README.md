# omerfaruksimsek.com

Personal website, blog, and portfolio of Ömer Faruk Şimşek.

Built with [Astro](https://astro.build) and deployed on [Netlify](https://netlify.com).

## Stack

- **Framework:** Astro 7
- **CMS:** Decap CMS (Git-based)
- **Deployment:** Netlify
- **Styling:** Custom CSS with dark/light mode

## Features

- Blog with web-based CMS panel (`/admin`)
- Projects portfolio with detail pages
- Dark / light mode toggle
- Responsive design
- Auto-deploy on push

## Local Development

```sh
npm install
npm run dev
```
