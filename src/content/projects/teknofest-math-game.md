This project focuses on accessibility in education. It uses high-contrast visuals to stimulate visual perception while solving math problems. Custom color palettes are implemented for different types of color blindness, ensuring inclusivity. The game provides a unique combination of education and health awareness.

**Technical Details**

* Engine: Unity
* Rendering: High-contrast UI system
* Accessibility: Custom color palettes (color blindness support)
* Gameplay: Math puzzle mechanics
* UX: Adaptive visual feedback system
