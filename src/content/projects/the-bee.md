The Bee is a first-person action game where players defend themselves against swarms of attacking bees using a ninja knife in a forest environment. It was developed in Unity 2020 as one of my first freelance projects at the age of 14.

### Key Features

* First-person melee combat.
* AI-controlled enemy bees.
* Wave-based survival gameplay.
* Forest environment.
* Responsive combat mechanics.

### Technical Details

Game Engine: Unity 2020

Programming Language: C#

Platform: Windows

Genre: First-Person Action (FPS)

Core Logic: AI enemy behavior, melee combat, player health system, and wave-based gameplay.
