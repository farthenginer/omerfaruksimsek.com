This project transforms history education into an interactive experience. It features both single-player and real-time multiplayer modes, allowing users to compete and learn simultaneously. The game achieved first place in the TÜBİTAK regional competition and advanced to the national finals in Ankara. It is also published on Google Play.

![](../../assets/uploads/0e0035c4-10a7-45cd-8c88-21851f16eb44.png)

**Technical Details**

* Engine: Unity
* Multiplayer: Photon Network (PUN)
* Backend: Firebase Realtime Database
* Architecture: Client-server multiplayer system
* Features: Real-time quiz mechanics, leaderboard system
