LOV is a tool designed to track changes in publicly available social media profiles.



With LOV, you can monitor updates such as username changes, bio edits, and profile activity without needing to log into an account.



Features:



\* Track username changes and view username history

\* Monitor bio updates and access previous versions

\* Follow profile activity, including profile picture and display name changes

\* Receive notifications when tracked information is updated

\* View changes in a chronological timeline

\* Get weekly summaries of tracked activity



LOV works exclusively with publicly available data. No login is required.



How it works:



\* Enter a public username

\* Start tracking profile changes

\* Receive notifications when updates occur

\* Review changes at any time



Privacy:



LOV does not require account access and does not collect private user data.
