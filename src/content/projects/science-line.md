Inspired by quiz games, this project includes a real-time multiplayer system where players compete in knowledge-based challenges. Built with Photon PUN and Firebase, the project was paused before reaching MVP stage due to shifting focus to other priorities.

**Technical Details**

* Engine: Unity
* Multiplayer: Photon PUN
* Backend: Firebase
* System: Level-based quiz progression
* Status: Prototype / Paused
