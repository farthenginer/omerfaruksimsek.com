School Runner 3D is an educational endless runner game where players must follow school rules and safely reach the school bus while avoiding obstacles and other students. Developed with Unity, the game was created as part of a PhD research project, where player data was collected through Firebase Authentication and Firebase Realtime Database. The project was successfully used in academic research and contributed to the supervising teacher earning their PhD degree.

### Key Features

* Educational gameplay focused on school rules.
* Firebase Authentication for player accounts.
* Online leaderboard using Firebase Realtime Database.
* Real-time gameplay data collection for research.
* Endless runner mechanics with progressively challenging obstacles.
* Interactive tutorial before gameplay.
* Student and obstacle avoidance mechanics.

### Technical Details

Game Engine: Unity

Programming Language: C#

Backend: Firebase

Authentication: Firebase Authentication

Database: Firebase Realtime Database

Platform: Windows / Android

Genre: Educational Endless Runner

Core Logic: Endless runner mechanics, player authentication, online leaderboard, gameplay analytics, and research-oriented data collection.
