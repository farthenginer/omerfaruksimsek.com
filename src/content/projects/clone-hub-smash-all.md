Clone Hub - Smash All is a hyper-casual mobile game where players eliminate waves of enemies using spinning blades and unlock powerful abilities to defeat dozens of opponents simultaneously. Players collect gold during gameplay to purchase new skills and upgrades, creating a fast-paced and satisfying action experience.

### Key Features

* Fast-paced hyper-casual gameplay.
* Multiple unlockable blade types.
* Special abilities capable of eliminating large groups of enemies.
* In-game shop for purchasing upgrades with collected gold.
* Various enemy types and increasing difficulty.
* Endless score-based progression.
* Responsive visual effects and combat feedback.

### Technical Details

Game Engine: Unity

Programming Language: C#

Platform: Android

Genre: Hyper Casual / Action

Core Logic: Endless gameplay, enemy spawning, ability system, in-game currency, upgrade shop, and score progression.
